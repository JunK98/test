# READ ME 정리

1. 제목
제목은 "#" 갯수로 구분하며  1개(제일 큼)~6개(제일 작음)

---

2. 가로 줄 넣기
가로줄은 "---' 또는 "***"


3. 목록정리
  - 순서 있는 애는 '1. ' 2. ' 이런 식으로 정리
  - 순서 없는 애는 '+ ' '- ' '* '  들여쓰기가 필요할 경우 탭 누르기

4. 텍스트 꾸미기
  - 기울기: '*~*' 또는 '_ ~ _' 로 감싸기
  - 굵게: '** ~ **' 또는 '__ ~ __' 로 감싸기
  - 굵은 기울기: '*** ~ ***' 또는 '___ ~ ___'로 감싸기
  - 취소선: '~~~ ~ ~~~' 로 감싸기

5. 소스코드 넣기
  - "`~`" (백틱)으로 감싸기
  - 한 줄 코드: (백틱) " `` " 으로 감싸기
  - 여러 코드: (백틱3개+언어 이름)으로 감싸기 "  

ex) ```python
<script> print(hello)</script>```
